Important note: the machine is using Arch Linux, which is notorious for being difficult to manage (but has some nice pros too!).
It was a pain in my ass. It shouldn't have been, but it was. I'm going to tell you the result of this process before I get into the nitty-gritty of it because I think it will be more interesting and informative that way.

The Results

The results were as follows:

  • A proper static IP address
  • A functioning routing table
  • Access to DNS nameservers
  • A port-forwarded Arch Linux server
These results might not seem like much, but they were hard fought for.
